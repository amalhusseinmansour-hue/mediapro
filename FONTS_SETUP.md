# تعليمات تثبيت خط Noto Kufi Arabic

## الخطوات:

1. قم بتحميل خط Noto Kufi Arabic من Google Fonts:
   https://fonts.google.com/noto/specimen/Noto+Kufi+Arabic

2. قم بتنزيل الملفات التالية وضعها في المجلد `assets/fonts/`:
   - NotoKufiArabic-Regular.ttf
   - NotoKufiArabic-Medium.ttf
   - NotoKufiArabic-SemiBold.ttf
   - NotoKufiArabic-Bold.ttf

3. بعد وضع الملفات، قم بتشغيل:
   ```bash
   flutter pub get
   flutter clean
   flutter run
   ```

## ملاحظة:
حتى يتم تحميل الخطوط، سيستخدم التطبيق الخط الافتراضي للنظام.
