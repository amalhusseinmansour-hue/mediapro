## Upload Steps Summary

1. Go to https://play.google.com/console
2. Create new app → Select "App" → Free
3. Fill in App details (name, descriptions)
4. Upload app icon and feature graphic
5. Upload screenshots (minimum 2)
6. Go to Production → Create release → Upload APK
7. Complete Content Rating questionnaire
8. Complete Data Safety form
9. Set pricing (Free with IAP)
10. Submit for review

Review typically takes 1-7 days.
